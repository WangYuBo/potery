# 现代诗生成器

一个基于纯前端技术的现代诗创作工具，用户可以通过点击随机字词来创作现代诗。

## 功能特点

- 🎯 点击随机字词创作现代诗
- 📝 支持多行创作，可自由换行
- 🖱️ 支持拖拽调整诗句顺序
- 📋 一键复制生成的诗歌
- 📱 完全响应式设计，适配各种设备
- 🚀 纯静态部署，支持 GitHub Pages

## 技术架构

- **前端框架**: 原生 HTML5 + CSS3 + JavaScript
- **响应式设计**: CSS Grid + Flexbox + 媒体查询
- **交互功能**: 原生拖拽 API
- **部署平台**: GitHub Pages
